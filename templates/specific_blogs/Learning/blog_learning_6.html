How I learned to code

I‘ve build a roadmap that you can see right here.

Roadmap

In march 3 2021 I decided that I want to learn to code because I wanted to create a Minecraft plugins for the server that we were playing during quarantine.

So I learned Java and how to code Minecraft plugins. (I had no idea what I was doing back then) I just followed tutorials and tried to change something and make it work I had no idea what I was doing.

After about one week I decided that I will give it a try. So I started learning java from the beginning.

I learned some basic programming concepts like loops statements variables etc ..

After about a month I decided that I will start building the app idea that I wanted to build but I find out that I am not capable of building something like that. (I try to develop my billion-dollar app idea)

After I failed in my journey of becoming a rich app developer I tried web development for the first time in about 6 months of coding.

I’ve learned how the web is structured and I was fascinated by the technologies that are included in the web. So I checked a few websites about the technologies and I choose to use Flask as my first web framework it is one of the easiest and also it uses Python and I wanted to learn Python.

I was learning only backend development I don’t understand HTML and CSS and Javascript and I’ve got a big learning hole I was able to do CRUD with database and I didn’t understand how does the frontend work. How the HTML page is structured I only know how to display data in Jinja.

After I got tired of being so stupid in frontend I discovered that I pretty much enjoy it. So I build a website in HTML and CSS and with Flask on the Backend. I was now able to build a pretty good looking website but I had no idea how does the javascript works. When I was doing something in javascript I just copy and paste it in most cases but after some time I wanted to create something more custom and I tried to learn it.

When I started learning javascript I wasn’t really good in it I understand just a basics and after some time I decided that I will build few beginner projects. These were calculator, Pomodoro timer, Form validation, Random advice generator.

After I started getting decent understanding of javascript I decided that I will move to the React.js library I am also not that good in design so I started taking screencast tutorials on scrimba.

Besides of my technological learning journey I was building core related projects for the technologies that I was learning. I’ve build a website where you can learn computer networking, and other projects that you can see in my portfolio.
