How I learned Frontend

When I started learning frontend I can say that I was educated and have a solid understatement of how my brain works and how I can learn anything fast. (It was in about 8 months of my coding journey)

1. HTML, CSS

in my first one or two months in learning frontend, I was just doing HTML, CSS and Python flask. I’ve got a solid understatement of how to work with dynamic data in Flask. The library for that was called Jinja2 it is a little bit similar to React in some ways.

I’ve built a few simple and one of them was the Networking App where I implemented my knowledge of SCSS Python and HTML

2. SCSS, Javascript Basics.

After I’ve got a solid understatement of CSS and HTML I started learning Javascript.

In this part of my learning journey, I’ve built my portfolio website.

I’ve learned how to work with media queries and responsive design I also get a solid understanding of modern CSS features

3. Advanced CSS, Javascript

After I’ve built my portfolio, I find out that I don’t understand CSS properly so I started learning how to implement different combinations of CSS properties like positions size units, etc...

- Besides that, I’ve built this blog website.

4. React.js, ES6

- When I started feeling confident in javascript and my CSS skills. I decided to learn my first javascript library I choose react for the ecosystem that is built around it.

- I’ve started taking classes on Scrimba
