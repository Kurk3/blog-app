Project based learning

When I started coding I wasn't really happy with the progress that I’ve made I did some simple problems but I don’t enjoy it at all so I started doing more project-based learning and I would like to show you how I learn from it.

Steps :

- Choose a project that I want to build

- Create a list of things that I need to learn before starting the project

- Understand the basics of the technologies that I will use

- Start building the project

Important step :

When I am working on some feature and I have no idea how to do it I spend a lot of time learning about that topic when I know that I will use this knowledge in the future.

I call this technique dopamine loop when I can’t solve the problem I rather spend time learning something useful and get the motivation back and work on that feature again and when I fail again I do this cycle again, and I will stop doing this until I solve the problem and do the feature.
