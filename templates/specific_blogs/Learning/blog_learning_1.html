Fast Learning

In this article, I will describe the steps that help me learn things faster.

1. Gather resources and information about the topic and do the proper research.

When I started my coding journey I had no idea where should I start so I started researching for the best curriculum that can teach me Java. When I look back at that point I think that I don’t spend enough time researching and checking forums If I do that I will not have that many learning gaps.
so do the proper research

2. Create a structured learning plan for the specific topic

if you find the best resources for that specific topic that you want to learn, you should create a structure where you will put every topic you need to learn before moving to another one. The best application for making this type of structure is Notion where you can track the things that you learned.

3. Learn a little bit of theory and instantly apply

To be more efficient in learning you should not read or watch the thing that you want to learn you should practice instead if you know enough theory so If you learn something automatically apply it.

4. Build projects and document your journey

If you want to learn for example coding you should document and take notes from what you learned. This can be applied to any type of industry when you document or put things in one place you will be much more satisfied with the work that you’ve done.
